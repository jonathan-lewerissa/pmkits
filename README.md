# PMK ITS

## Official Website Development for PMK ITS
_Website yang dibuat untuk Persekutuan Mahasiswa Kristen (PMK) ITS_

## Instalasi awal
```
php artisan migrate
php artisan migrate --database="frontdb" --path="database/migrations/frontdb"
php artisan db:seed

php artisan key:generate
```
